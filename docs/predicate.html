Assumes the variables in \(P\) are of the form <var_name>’@’<number> Will return a function that acts as a predicate that can be indexed. For example, suppose we create T ≡ x@1 = x@0 + 1, then T(i) ≡ x@{i+1} = x@i + 1

predicate.instantiate_predicate_variables_at_plus(P, i=0)[source]¶

Assumes the variables in \(P\) are of the form <var_name>’@’<number> Will return a predicate with variables indexed using i. For example, suppose we create T ≡ x@1 = x@0 + 1, then x@{i+1} = x@i + 1 is returned.

In general, given if x@k is in P, then it will be mapped to x@{k+i}

Return type

FNode

Returns

Predicate new_P such that for each x@k variable in P will correspond to x@{k+i} in new_P.
